# QA Automation Technical Test

## Overview
This repository contains automated tests for the Shift Management API.

## Features
- Complete E2E API test suite using Postman/Newman
- CI/CD integration with GitHub Actions
- Comprehensive test coverage including authentication, authorization, and shift management

## Quick Start
1. Clone the repository
2. Install dependencies: `npm install`
3. Run tests: `npm test`

## Project Structure
- `/postman` - Postman collections and environments
- `/.github/workflows` - CI/CD pipelines
- `/TESTING.md` - Detailed testing documentation
